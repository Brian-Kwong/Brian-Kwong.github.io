This package contains the source code for the frontend of the website using React.js and Vite

If you would like to run this project locally, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the `brian-kwong` package directory:

```bash
cd packages/brian-kwong
```

3. Install the necessary dependencies using npm or yarn:

```bash
npm install
# or
yarn install
```

4. Start the development server:

```bash
npm run dev
# or
yarn dev
```

When ready to build for production, run:

```bash
npm run build
# or
yarn build
```

The page is automatically deployed to GitHub Pages upon pushing changes to the main branch.
